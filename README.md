# Connectivité Internet en Afrique — Analyse 2022-2023

## Description
Ce projet analyse la pénétration d'internet dans 50 pays africains
à partir de données réelles. L'objectif est de comprendre quels pays
sont les plus connectés, lesquels sont en retard, et où se situe
la Côte d'Ivoire par rapport au reste du continent.

## Questions auxquelles le projet répond
- Quels sont les 10 pays africains les plus connectés ?
- Combien de pays africains ont moins de 20% de connectivité ?
- Où se situe la Côte d'Ivoire par rapport à la moyenne africaine ?
- Comment les taux de pénétration sont-ils répartis en Afrique ?

## Résultats clés
- Le taux moyen de connectivité en Afrique est de **42.5%**
- Le Maroc (89.9%) et la Libye (88.4%) sont les pays les plus connectés
- 10 pays africains ont moins de **20%** de connectivité
- La Côte d'Ivoire se classe **23ème sur 50 pays** avec 38.4%
